# Laboratorio 1:

## Instalation: 

```
 cd /usr/local/lib    
 sudo curl -O https://www.antlr.org/download/antlr-4.10-complete.jar
 export CLASSPATH=".:/usr/local/lib/antlr-4.10-complete.jar:$CLASSPATH"
 pip install antlr4-python3-runtime antlr4-python3-runtime==4.10
```

## Run:

```
    make all
```


## Clean:

```
    make clean
```
